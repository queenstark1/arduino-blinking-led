# Blinking LED Project

## Description
This is my first Arduino project using Tinkercad.  
The LED blinks on and off every second.

## Components Used
- Arduino Uno
- LED
- 220Ω Resistor
- Wires

## How It Works
The Arduino sends a HIGH signal to turn on the LED and LOW to turn it off with a delay of 1 second.
